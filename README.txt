This algorithm was written in python,it implements pandas and numpy libraries to perform matrix operations.
(Make sure you install it in your python).

The column profile_id represent a profile
The column other_profile_id represent a other profile
The column score  is a mathematical expression of how happy a profile with other profile… 

You can rum the project opening pycharm and run or using a terminal and write python okcupid.py in the folder container the file okcupid.py
